四.可能遇到的一些问题及解决方法

1.Hexo报错 ERROR Deployer not found: git

这是因为没安装hexo-deployer-git插件，在网页源码根目录下打开命令行输入：

之后再输入hexo d部署就行了。

2.Hexo上传Git库速度陡降

可能是因为多次生成产生了冗余。

使用hexo clean清除缓存，如果删除时卡住，请手动执行：找到网页源码根目录下的 “public” 与 “.deploy_git” 文件夹，将其删除后重试。

3.上传后Git库不更新

由于Hexo没有检测到更新，并没有进行推送操作，所以需要删除老版本：网页源码根目录下的 “.deploy_git” 文件夹。

4.Hexo报错Error：spawn failed

五.主题

建站后需要选择一个中意的博客主题，Hexo自带了300+款主题 （点击图片为示例，点击名称为下载地址），每一个都有示例网页。下载解压好后把文件放在根目录下的 “theme” 文件夹里（以一级文件夹的形式），并把Hexo的 “_config.yml” 中的 theme 项改为放入 “theme” 文件夹的主题文件夹名称：

PS：选主题很重要，选好后很多个性化需要更改主题的配置文件，每个主题的文件结构不同，再更改很麻烦。

一般主题会有自己的教程网站以帮助用户实现不同功能（具体请自行搜索），建议先完整浏览一遍，避免额外安装插件时与主题冲突（这种情况其实挺多的）。
